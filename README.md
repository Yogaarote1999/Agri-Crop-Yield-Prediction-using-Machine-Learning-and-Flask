# My Project
# Crop Yield Prediction (Machine Learning + Flask)

## Description
This project predicts crop yield based on weather and soil data.  
It helps farmers and agriculture planners make better decisions.

## Tech Used
Python, Pandas, Scikit-learn, Flask

## Features
- Data cleaning & preprocessing
- Machine learning model training
- Simple Flask web app for predictions

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Start app:
   python app.py


## Download the Model File

The model file is too large to store on GitHub.

Download it here:

https://drive.google.com/file/d/1fG-qh6Bfmw95PVHYWA_TcLu2zLG5dSAT/view?usp=sharing

After downloading, put the ZIP file inside the `data/` folder of the project.
